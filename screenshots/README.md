# Screenshots Directory

This directory contains all screenshot evidence for the WEB303 Practical 6 submission.

## Required Screenshots

Place the following screenshots in this directory:

1. **1-services-running.png** - Output of `docker compose ps`
2. **2-unit-tests.png** - Output of `make test-unit`
3. **3-integration-tests.png** - Output of `make test-integration`
4. **4-e2e-tests.png** - Output of `make test-e2e`
5. **5-api-gateway-grpc-logs.png** - API Gateway gRPC initialization logs
6. **6-order-service-grpc-logs.png** - Order Service gRPC communication logs
7. **7-all-tests.png** - Output of `make test-all`
8. **8-live-api-demo.png** - (Optional) Live API curl commands and responses
